Setting up the environment:
Pre-requisites:
*install java sdk  (java sdk)
*install nodejs
*install python 2
*install android studio

Installing react native:
*    npm install -g react-native-cli

To run: npm install
*on IOS: npx react-native run-ios
*on android: npx react-native run-android
